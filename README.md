# Eventflow

Eventflow is a lightweight event discovery and ticketing app for browsing events, registering attendees, issuing tickets, and validating entry.

## Features

- Browse featured and upcoming events
- Search events by title and filter by type
- View event details, schedules, locations, and RSVP activity
- Register attendees and generate unique ticket IDs
- Validate tickets directly from the app

## Stack

- React
- Vite (scaffolding + dev server)
- Language: JavaScript
- Tailwind CSS: Enabled
- Structure: Feature-based
- React Router
- MantaHQ SDK
